字数统计: 2,846字 | 阅读时长 ≈ 12分钟

作者：TeddyZhang

简介


　　本文主要是对类中出现的静态成员、常量成员函数（对象）、封闭类以及友元概念的总结。
　　静态(static)成员分为静态数据成员和静态成员函数。本质上是全局变量和全局函数，但在类中书写，显得与类紧密联系，但并不作用某一对象。
　　封闭类是指这个类中还包含着另一个类的对象。
　　常量对象与常量成员函数保证了一些数据和方法的安全性，可靠性。
　　友元可以提高程序的执行效率，引入了访问类中私有空间的另一种形式，不单纯通过类的成员函数。

发表于 2018-01-02 | 分类于 SOFTWARE CODING
字数统计: 1,683字 | 阅读时长 ≈ 7分钟

作者：TeddyZhang

简介


　　本文主要介绍C++中特殊的成员函数的知识及概念！
　　构造函数的作用主要是为对象的成员进行初始化，并为对象分配空间。名字与类名相同，可以用参数，但不可以有返回值，void也不可以。
　　思考一下，为什么不可以像结构(struct)那样初始化对象(class)，而要创建构造函数？很简单！C++中的结构和对象区别在于结构默认变量是public，而c++默认是private，不可以在外部直接访问，这就是C++的“数据隐藏”功能。
　　析构函数也是一种特殊的成员函数，主要用于释放分配给对象的内存空间，并做一些善后的工作。

哈夫曼(Huffman)编码（Python 3.6）

发表于 2017-05-22 | 分类于 DATA COMPUTING
字数统计: 2,202字 | 阅读时长 ≈ 8分钟

作者：TeddyZhang

简介

本文主要介绍使用Python实现通信过程中的信源熵的计算，并对其进行Huffman编码以及计算出编码效率。
哈夫曼编码是一种最常用信源编码，为最佳码，是可变长编码的一种。
这种编码方式于1952年由Huffman提出，依据字符在文件中出现的概率对字符进行唯一编码，并且保证了可变编码的平均码长最短，从而编码效率较高，
被称为最佳码。
