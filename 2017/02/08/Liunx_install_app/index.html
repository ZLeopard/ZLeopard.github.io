安装Python IDE Pycharm

PyCharm是一种PythonIDE，带有一整套可以帮助用户在使用Python语言开发时提高其效率的工具，比如调试、语法高亮、Project管理、代码跳转、智能提示、自动完成、单元测试、版本控制。此外，该IDE提供了一些高级功能，以用于支持Django框架下的专业Web开发。
同时支持Google AppEngine，PyCharm支持IronPython。这些功能在先进代码分析程序的支持下，使PyCharm成为Python专业开发人员和刚起步人员使用的有力工具。

安装VPN Lantern(蓝灯)

VPN，大家都懂，墙外虽好，但也要科学上网。我是个Google Fans，没有Google的Ubuntu根本就玩不嗨=￣ω￣=

安装方法

由于Lantern整个项目托管在github上并且是开源的，所以可以通过其进行下载。
项目地址为：https://github.com/getlantern/lantern
在其项目的README.md文件中查找ubuntu下的安装包，下载文件至相应目录。
ubuntu16.04的软件中心有bug，不可以直接安装.deb文件，所以我们在这里使用gdebi的方式进行安装。

实验验证

安装好了之后，直接在终端输入lantern,或到/usr/share/applications寻找图标打开，在浏览器会出现lantern界面，就可以愉快的访问墙外的世界了，哈哈哈~~
