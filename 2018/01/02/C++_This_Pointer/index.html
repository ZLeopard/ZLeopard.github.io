C++类的特殊成员函数（构造函数、析构函数）

发表于 2018-01-02 | 分类于 SOFTWARE CODING |
字数统计: 1,683字 | 阅读时长 ≈ 7分钟

作者：TeddyZhang

简介


　　本文主要介绍C++中特殊的成员函数的知识及概念！
　　构造函数的作用主要是为对象的成员进行初始化，并为对象分配空间。名字与类名相同，可以用参数，但不可以有返回值，void也不可以。
　　思考一下，为什么不可以像结构(struct)那样初始化对象(class)，而要创建构造函数？很简单！C++中的结构和对象区别在于结构默认变量是public，而c++默认是private，不可以在外部直接访问，这就是C++的“数据隐藏”功能。
　　析构函数也是一种特殊的成员函数，主要用于释放分配给对象的内存空间，并做一些善后的工作。

一般构造函数

　　对于一般的构造函数，主要用于初始化，如果自定义了构造函数，则编译器不会生成默认的构造函数，而且在对象生成
时被调用，对象一旦生成，就不能再运行构造函数。
　　一个类可以有多个构造函数，而且构造函数的参数可以使用默认参数或无参数，也可以进行函数的重载。

　　
结果为下面情形，注意函数重载时，匹配函数的情形：
Constructor 1 called
Constructor 2 called
Constructor 2 called
Constructor 2 called
Constructor 1 called

复制构造函数（Copy Constructor）

　　在C++中，除了一般的构造函数外，还有一中特殊的构造函数–复制构造函数，作用是：用一个已存在的对象来初始化该类的
新对象，用户可以根据需要定义一个复制构造函数，如果用户没有自定义，则默认编译器会生成一个没有任何内容的复制构造
函数。
　　自定义复制构造函数的形式为：
类名(类名&对象名)
{
函数体
}
其中对象名是用来初始化另一个对象的引用。
复制构造函数在以下的三种情况可以被调用：

  • 用类的对象去初始化该类的另一个对象时。
  • 函数的形参是类的对象，调用函数时进行形参和实参相结合时。
  • 函数的返回值是类的对象，函数执行完返回调用者时。

析构函数

析构函数作用时释放分配给对象的内存空间，并做一些善后的工作。在声明定义和使用时应注意：

  • 析构函数的名字必须与类名相同，但前面加上“~”
  • 没有参数，没有返回值，不能重载，在一个类中只能有一个析构函数。
  • 当撤销对象时，系统自动完成空间的释放和善后工作。（先创建的后析构）
