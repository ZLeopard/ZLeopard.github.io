简介


　　本文主要讲函数对象的概念及其用法，那么什么是函数对象呢？
　　函数对象顾名思义，肯定和函数有关，其实质就是，如果一个类在其成员函数中重载了符号”()”，那么这个类就称为函数对象。
　　为什么叫函数对象呢？那是由于调用时类似于函数的调用，但实际只是调用了类成员函数中重载”()”的那个成员函数。接下来用例子说明吧，Lets Go~
